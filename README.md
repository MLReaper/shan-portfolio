Personal Portfolio Website

Overview

This is my first project on Coursera. As part of the final project, I created a personal portfolio website using HTML, CSS, and JavaScript.

The project demonstrates the fundamental web development skills I learned throughout the course, including webpage structure, styling, responsive layout design, and basic interactivity using JavaScript.

Project Description

ABC Consultant is a reputed placement company based in Seattle that helps individuals with less than five years of experience find suitable employment opportunities.

As a web developer hired by ABC Consultant, I was tasked with creating a single-page portfolio website for one of their clients. The portfolio is designed to showcase the client's professional profile to prospective employers.

The website includes the following sections:

* About Me
* Skills
* Projects
* Recommendations

Features

About Me

* Profile image
* Name and professional title
* Brief introduction and career summary

Skills

* Technical skills with logos/icons
* Years of experience displayed for each skill

Projects

* Three unique projects showcasing development experience
* Project descriptions displayed in a structured format

Recommendations

* Three custom recommendations
* Ability to add new recommendations dynamically using JavaScript

Additional Functionality

* Interactive navigation bar with hover effects
* Home icon for quick navigation to the top of the page
* Confirmation pop-up message when a recommendation is submitted

Technologies Used

* HTML5
* CSS3
* JavaScript

Learning Outcomes

Through this project, I gained hands-on experience in:

* Structuring web pages with HTML
* Styling user interfaces with CSS
* Creating interactive features with JavaScript
* Manipulating the DOM
* Building a complete single-page portfolio website

Evaluation Criteria

The project was developed according to the following requirements:

1. Styled name displayed on the top-left of the page.
2. Navigation bar containing About, Skills, Projects, and Recommendations sections with hover effects.
3. Unique About Me section with profile image and personal summary.
4. Skills section displaying logos and skill information in the required format.
5. Three custom projects different from the sample provided.
6. Three custom recommendations different from the sample provided.
7. Ability to add new recommendations dynamically.
8. Home icon for returning to the top of the page.
9. Confirmation pop-up displayed after submitting a recommendation.

Conclusion

This project marks an important milestone in my web development journey and represents my first completed project on Coursera. It allowed me to apply the HTML, CSS, and JavaScript concepts learned throughout the course to build a functional and interactive portfolio website.
